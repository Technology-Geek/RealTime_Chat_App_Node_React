## Install Dependencies

Client:

```bash
cd ./client
npm install
```

Server:

```bash
cd ./server
npm install
```

## Quick Start

Client:

```bash
cd ./client
npm start
```

Server:

```bash
cd ./server
npm start
```

Developer Server:

```bash
cd ./server
npm run dev
```
